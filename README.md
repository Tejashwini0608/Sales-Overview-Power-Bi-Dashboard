📊Sales Overview Dashboard (Power BI)
📌 Project Overview
This project presents a dynamic Sales Overview Dashboard built using Power BI, designed to analyze and track key business metrics across different regions.
The dashboard enables users to:
*Monitor Sales, Profit, and Quantity
*Compare Current Year (CY) vs Previous Year (PY)
*Analyze performance across Central, East, South, and West regions
*Gain insights through interactive visualizations

🎯 Business Requirement
The goal of this project is to design a data-driven dashboard that:
*Tracks key KPIs (Sales, Profit, Quantity)
*Supports year-based filtering
*Allows dynamic metric selection
*Provides year-over-year comparison
*Helps stakeholders make informed business decisions

📊 Key Features
✅ KPI Metrics
*Sales
*Profit
*Quantity
🔄 Dynamic Metric Selection
Users can switch between:
*Sales
*Profit
*Quantity
📅 Year Filter
*Select any year to view performance
*Automatically updates all visuals
📈 Year-over-Year Analysis
*Current Year (CY)
*Previous Year (PY)
*YoY Growth (increase/decrease)

🌍 Regional Analysis
The dashboard provides insights for:
*Central Region
*East Region
*South Region
*West Region
Each region includes:
*KPI values based on selected year
*Previous year comparison
*Monthly trend using bar sparklines
*Average line for better analysis

📌 Visualizations Used
🗺️ Bubble Map
*Displays Sales by State
*Bubble size represents sales volume
📊 Bar Chart
*Sales comparison across states
*Sorted for easy analysis
📉 Sparklines
*Monthly trends visualization
*Includes average line
📋 Data Table
Includes:
*CY Sales, PY Sales, YoY Sales
*CY Profit, PY Profit, YoY Profit
*CY Quantity, PY Quantity, YoY Quantity

⚙️ Tools & Technologies
*Power BI
*DAX (Data Analysis Expressions)
*Data Modeling
*Data Visualization
