# 📊 Sales Overview Dashboard (Power BI Project)

## 📌 Project Overview
This project focuses on analyzing sales and profit data to identify key business trends and support decision-making. The dashboard provides a clear view of sales performance across different regions, products, and time periods.

---

## 🎯 Objectives
- Analyze overall sales and profit performance
- Identify top-performing products and regions
- Track monthly sales trends
- Highlight areas with low profit

---

## 🛠 Tools Used
- Power BI
- Excel (Dataset)
- Data Cleaning & Transformation

---

## 📊 Dashboard Features
- KPI Cards (Total Sales, Total Profit, Quantity Sold)
- Sales trend over time (Line chart)
- Region-wise sales performance
- Top products by revenue
- Interactive filters (slicers)

---

## 🔍 Key Insights
- Certain regions contribute higher sales but lower profit margins
- A small number of products generate most of the revenue
- Sales show noticeable growth in specific months

---

## 📁 Files Included
- Sales Dashboard (.pbix)
- Dataset file

---

## 🚀 Conclusion
This dashboard helps businesses quickly understand their sales performance and make data-driven decisions.
